How to start the frontend and backend

First navigate to the WEB701-ProjectFinal folder location in either VS Code or in command prompt
Open two terminals
In first terminal type in 'cd backend' followed by 'npm start'
In second terminal type in 'cd frontend' followed by 'npm run serve'

NOTE - login system doesn't function correctly so after logging in, refresh the page to 
update the user state. Same goes for logging out.
